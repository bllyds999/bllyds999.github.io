我之前不是写过一篇文章，吐槽过我的旧电脑升级 Win11 后的各种问题吗？那台电脑本来运行 Win10 还算流畅，结果一升级到 Win11，开机速度直接翻倍——不是变快，是变慢了一倍。任务栏时不时卡死，右键菜单要等两秒才弹出，连资源管理器都学会了“未响应”这门艺术。更别提那强制合并的任务栏图标，对多任务操作简直是灾难。哪怕后来我换了性能强劲的游戏本，按理说硬件足够碾压任何系统，可我依然下意识地第一时间安装了其他系统，最次也是受不了 Linux 那无休止的驱动折腾和命令行依赖后，又灰溜溜地转回了熟悉的 Win10 系统。那种感觉就像是被 Windows 伤了心，却又找不到完美替代品的游牧民族。

直到现在，我终于用上了 macOS，才算暂时找到了一个安静而稳定的栖身之所。深夜的时候，人总是容易陷入一种放空的状态，手指机械地滑动着屏幕，刷着一个又一个短视频。就在这样的时刻，我刷到了一个标题颇有吸引力的视频，内容是关于如何关闭电脑上那些烦人的广告。视频里的博主神情十分正式，对着镜头一本正经地说——如果你升级了 Win11（你接下来是要讲性能优化的问题，还是系统卡顿的解决方案？）——那么你的锁屏界面、搜索界面、小组件界面都会出现广告，而我现在就来教你如何把它们一个个关闭掉。他的语气像是在处理一件严肃的系统底层问题，而我则抱着一种“我倒要看看有什么高招”的心态继续看了下去。

出乎意料的是，作为一个前 Linux 用户、现 macOS 用户，我大多数时间都活在 UNIX 那种干净、克制、命令行至上的温床里，脑子里早已习惯了“系统就是工具”的朴素认知。当我看到那个标题时，第一反应居然是你是不是要教我怎么调优系统性能、关闭后台进程、修改注册表来提升响应速度——完全是 Linux 时代留下的“性能焦虑”后遗症。我完全没想到电脑里居然有“广告”这一层，这个词出现在操作系统语境里，对我来说就像是在图书馆里听到广场舞音乐一样荒诞。它真的是在我的意料之外，以至于我愣了几秒才反应过来：哦，原来 Windows 用户日常还要对抗这种东西。

倒不如说，为什么我的电脑，我花钱买的电脑，为什么要有广告在这里？这明明是一台硬件设备加上一份正版操作系统的组合，我付过钱了，我是它的主人，而不是它用来向第三方展示我注意力的橱窗。我以前用 Windows 的时候，怎么就没觉得这些广告冒犯和吵闹呢？现在回头想想，那时候的我甚至觉得这些推送是一种陪伴——就像是一个人在空荡荡的电脑世界里，突然有推送弹出来告诉你天气、新闻或者某款游戏在打折，像是透过一扇窗户看到了外面的热闹人间。那种孤独中的一丝喧嚣，反而让我觉得自己还连着网、还属于这个数字社会。

但我用不超过 10 秒的时间就接受了这个事实，就像以前的自己一样，习惯性地选择了“算了”。老实说，我对那些突然弹出来的广告弹窗，尤其是当你聚精会神写代码、剪视频或者打游戏打到关键时刻时，屏幕角落突然蹦出一个闪烁的推销界面，那种感觉真的是一种赤裸裸的冒犯和侵入——像是在你专注的脑海中硬塞进一块广告牌。但如果只是软件界面、网站页面里内置一些广告横幅，我顶多会觉得这是他们要维持运营、获取收益的正常手段，可以理解，也可以忽略。可操作系统不一样，它是所有软件运行的基础，是我与电脑交互的底层语言，不该成为广告的分发渠道。

我算是知道 UNIX 系统的好处了，那种“没有消息就是好消息”的设计哲学，让我终于能在数字世界里获得一片安静。虽然目前接受 Linux 还需要一个 Wine 的时间——毕竟有些 Windows 专属软件还是离不开，得靠兼容层去跑，时不时还要忍受配置失败的崩溃感——但总归是有个好看的界面和一片自由的、让人安静的空间。不会有人在锁屏时推荐你玩游戏，也不会在搜索栏里暗示你下载某款软件。我可能现在唯一能接受的广告，就是打开某个开源软件后，屏幕上缓缓弹出的那个“为本项目捐赠”的提示窗吧，因为它真诚、克制，而且你可以选择关闭，也可以选择感激。
