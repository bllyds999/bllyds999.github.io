去年九月份，我终于告别了那所令我痛心的技校。离开之后，日子一下子空了下来，整天在家里闲着没事干，心里总觉得不是滋味。为了给自己找点事做，也为了学点新东西，我便在抖音便宜网购了一台二手 MacBook Air，然后在小黄鱼上开了一家“技术小铺”。取名“热爱电脑的川川”，一方面是想告别那段并不愉快的技校时光，另一方面也是想用初中请假在家自学的 Linux 技术，在这个小小的网络角落里，重新开始做点有意义的事情。

我当时盘算得很简单：Linux 的用户那么少，而且看起来大家好像都是技术宅，应该不会有人想到在闲鱼上开店卖 Linux 相关服务吧？毕竟 Linux 系统使用中会遇到那么多难题，装个驱动、配个输入法、搞定无线网卡，哪一件不让人头疼？在我看来，那些能搞定 Linux 的人，平时大概只会在各种技术论坛里潜水和讨论，根本不屑于到闲鱼这种地方来做生意。于是，我便觉得自己找到了一个冷门又稳妥的方向。

现在回过头来看，我当时定的价格实在是太傻了。装一个 Arch Linux 只卖五块钱，装一个 Gentoo 也只收五块钱，就连那种最容易让人卡住、最折磨人的输入法安装，我也只敢要三块钱。我完全没考虑过自己的时间成本和精力成本，只是凭着一种“反正闲着也是闲着”的心态，拍脑袋就把价格定了下来。然而正是这套现在看来“无知”到离谱的定价策略，意外地为我带来了很大的收益。我陆陆续续接到了很多客户，有的是找我帮忙安装各种 Linux 发行版的，有的是想让我帮忙部署 Docker 容器的，还有的是系统出了莫名其妙的毛病，希望我远程帮忙排查问题的……一时间我的小铺竟然像模像样地运转了起来。

小黄鱼就是一个小小的线上市场，但社会上的三教九流、各色人等都能在这里遇到。来找我的人里，有对技术充满热情的中学生，有正在学运维的职校生，有在大学里专门学过运维的本科生，还有已经工作的在事业单位上班的人、做律师的、搞财务的，都成了我的客户。这让我第一次真切地感受到，原来 Linux 并非只是少数极客的玩物。在这些形形色色的客户中，给我带来最大看法和触动的是大学生和事业单位的人。

我以前总觉得，大学应该是一个能学到很多东西的地方，虽然网上经常说什么“清澈愚蠢的大学生”，但我看到这种说法时都是笑笑，觉得有些夸张了。可真正接触下来，我才发现有些事情比我想象的要复杂得多。让我印象最深的是，有一个来找我的大学生，她所在的班级里，全班同学居然都不会用 Docker。她亲口跟我说，她好几个朋友也完全不会写这些技术配置，甚至连概念都搞不清楚。听到这里，我忍不住笑了——原来大学里真的是什么人都有。

有那种认认真真学习、一路读到博士的学霸，也有这种上课就是混日子、考试全靠突击的。那些认真准备考研、努力读博、拼命考编的人，基本上不会在网上参与什么技术讨论。为什么呢？因为人家平时可能忙着背书、做习题、准备英语多少级的考试、考各种资格证书……时间早就被排得满满当当，根本没有精力在网上参与那些闲散的技术讨论。而那些热衷于在网上参与讨论的大学生，大多数都是一些在学校里比较松散的人。他们上课时习惯拿着手机不听课，课余时间又不知道怎么安排，于是就在网上到处说自己的情况，然后和同样处境的人抱团取暖，形成一个典型的沉默的螺旋。时间一长，这种群体形象就有了：好像所有大学生都不过如此。

其实也不能全怪他们，就像我在技校里看到的一样：如果他们上的课比我在技校里看到的还要水，周围坐着的同学也都在玩手机的情况下，他们也控制不住自己的。

事业编那边也有几种典型的人：公务员、国企员工、军人。其中有两位公务员大姐姐给我留下了特别深刻的印象。一位是真心想尝试用 Deepin 系统的，另一位则一心想着在上班时玩忽职守、偷偷玩游戏。同样是公务员，同样是面对操作系统，她们的态度和目的却截然不同。先说那位想尝试 Deepin 的大姐姐吧。那天她找到我的小店，委托我帮她下载系统。我帮她下载完、简单介绍了安装方法之后，她高兴地用上了。她告诉我，因为工作原因她那段时间要用 UOS，所以想先通过 Deepin 来习惯一下这类国产操作系统的操作逻辑。

另一位公务员大姐姐则完全相反，她非常讨厌 UOS。为什么呢？她跟我说，用这个电脑感觉特别不自由，什么都不能随便装，想在电脑上玩点小游戏都做不到。她自己尝试了好几次，发现根本没法玩，因为没有 ROOT 权限，什么都装不上。一开始她只说没法玩游戏。我听了之后傻眼了。这种人想要在上班时间玩电脑也就算了，用的居然还是单位的电脑，真把单位当成网吧免费供应电脑了吗？我都不知道她是怎样考上事业编的，单位还给配了这种系统。她跟我说，她们单位是一个非常小、非常透明、没人盯着看的那种小单位，让我不用紧张。我才不信呢，我们这边事业单位基本上都是 Win7，而她那“透明小单位”都用 UOS 了。

除了这两位形成鲜明对比的公务员姐姐，我还遇到了一位军人哥哥和一位国企的叔叔。那位军人哥哥找到我时，很着急地说，他们单位内部的邮件系统网页出了故障，用的是麒麟系统，自带的浏览器怎么也打不开那个页面。我就把他需要的浏览器安装包打包发给他，结果他运行之后，出现的问题还是那个。后来这件事就不了了之了。过了一段时间，他主动在微信上联系我说，他们单位后来统一装上了 Wine，用 Wine 运行 Windows 版的浏览器后，邮件系统就好了。我寻思这件事本身可能也跟我没什么关系，毕竟这是他们的内部系统，而且从 Linux 的角度来看，装个软件动不动就需要 ROOT 权限，权限太高的话，一旦出问题，损失对我来说太大。

那位国企的叔叔需要在 UOS 上运行 Docker，但他只是一个比较熟悉 C 语言的老手，对 Docker 这类容器化部署的工具并不了解。他自己折腾了两天都没有搞定，最后找到了我。我又花了不少时间远程指导，最终成功地在他那台“长城电脑”上部署好了 Docker。那也是我第一次亲身体验所谓的“长城电脑”。值得一提的还有一个小插曲：当时我离开技校已经好几个星期了；有一天，原来的班主任突然打来电话，说班里有人查出得了肺结核，要求我也去医院打疫苗做预防；那天我正好约了那位国企叔叔解决问题，于是整个人就是在医院里一边排队，一边打电话指导他操作，还得用手机远程操控他的电脑。说实话，用手机操控电脑，真的是不自在，我以前玩云电脑的时候怎么没感觉呢？

就这样，后来小店的生意渐渐淡了下来。订单越来越少，最后我干脆把店铺也关了。有些商品还被平台以各种理由判定为违规而下架，可是同样的商品、同样的表述，别人却可以正常挂在那里卖。这让我感到很是不解，慢慢地也就放弃了再开店的念头。虽然那段时间生意好的时候，我每天的零花钱都能赚到一百块钱左右，有时候还能给爸爸发两百块钱补贴家用，但是身体却渐渐撑不住了。最后因为生病，我终于彻底关闭了这个早已没什么人光顾的小店。我想你们一定觉得自己的 Linux 技术非常高超，什么问题都能解决，但那是相对于你们自己而言的。我们这种技术宅，一旦开始给别人解决问题，就会发现事情完全不是想象中那么简单。

这也是我头一次觉得，Linux 给普通人用，真的是不合适。
