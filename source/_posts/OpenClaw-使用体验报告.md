是的，我也用上了 OpenClaw，还特地发动态说明了前因后果。OpenClaw 的公开已经是去年的事情了，甚至今年早些时候还火过一回，但是我现在才用。并不是说我有多落后，反倒是我注重安全，所以才先观望一阵子再使用。现在知道了那么多缺点，再根据我现在的体验情况，来实锤几个事情。

## OpenClaw 很难安装

![](/assets/images/illustrations/openclaw-1.webp)

直接安装确实很难安装，但是通过 1Panel 等一些图形化工具安装可以特别简单，就跟你去应用商店下载软件一样，再加上现在各大博主也分享了自己做的一些图形化安装的应用，从安装到使用可以特别简单。

![](/assets/images/illustrations/openclaw-2.webp)

不仅可以一键安装，还可以图形化接入微信，非常省事了！这就是所谓的“早买早享受，晚买有折扣”，那些人上闲鱼花的几百块钱装这玩意？甚至都不需要上门安装，反倒是那些几块钱的良心商家真想帮忙的没人找。这里并不是说夸赞他们，我只是说，心急吃不来热豆腐，道理就是那么个道理。

## OpenClaw 很费 Token

这事是真的，你们可以看我的 DeepSeek 网页端后台：

![](/assets/images/illustrations/openclaw-3.webp)

平时我都不怎么用 DeepSeek API，用了花的也很少，今天想起来还有余额，于是就拿给龙虾用了，没想到一天就花了 3 块钱！我才问了几个问题啊，它自己就消耗了那么多。如果正常接入 Claude Code、Codex 那些，也不会花那么多吧？但是呢，也没有网上说的一天内扣光你银行卡那么离谱，因为这玩意只花你冲进去的 Token 费用，顶多第二天给你清零了。

除非你把自己都给卖了充钱给模型厂商，那你还是当我没说过吧！

### 罪魁祸首居然是它

![](/assets/images/illustrations/openclaw-4.webp)

是的，经过我的调查，它默认开启了 46 个 SKill，每个 Skill 的 Description 就很长。你们可以想象一下，每次对话除了你说过的话，还额外发了 46 句话，花的 Token 可不多吗？可以仔细看一下，这些全都是国内用不到的，你真的可以把这些 Skill 关了去下个你需要的 SKill，比如多搜索引擎，把它当作一个图形化、有记忆的 Claude Code。

## OpenClaw 用起来很简单

这句话我不敢苟同，因为我是一个连 iFlow 和 Trae Solo 都无法驯服的男人。我根本无法理解这些软件到底如何使用，因为它们实在是太黑箱了！对于 OpenClaw 一句话就可以开发项目，抱歉，我没那么多钱陪你玩，我只想要一个贾维斯……一个智能助手，一个智能助手！

![](/assets/images/illustrations/openclaw-5.webp)

因为好多 Skill 都已经被关掉了，人工智能现在啥也不会，导致只能当作一个普通的个人助手了，要不然真的很费钱。不过接入微信里面当作一个小姑娘调戏一下还可以，哈哈哈哈哈，这简直就是花钱找陪聊，甚至还不如陪聊。

## OpenClaw 适合你用吗

你能问出这句话，说明你真的需要观望一下是否要用。因为这玩意形式上不同于以前以往的小助手，更像是一个长期有陪伴性质的东西。不要听着贾维斯那么科幻的名字，人钢铁侠还是世界首富加天才大脑……说白了，普通人根本用不上 OpenClaw，给你个豆包说不定不会用呢，你还指望用这个需要自己部署，还花自己钱包的？
