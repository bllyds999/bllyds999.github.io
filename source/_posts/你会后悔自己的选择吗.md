先问几个问题，你会后悔自己的选择吗？到底有多后悔？后悔的是什么问题？人为什么会后悔，在如今的条件下，生活过得不如想象里当初选择的另一条路，觉得如果选择了另一条路会更好；现在的自己错过了当时本应得到的机会，内心里感觉懊悔不已；当时的自己说错话，导致错过了贵人……这些都是导致大部分人产生“后悔”的主要原因。

我从去年 5 月份拥有自己的第一个博客网站，今年也是 5 月份了。时隔一年，我的网站从理论上来讲的确达到了，大部分博客活动的运行标准——运行存在一年。我本应有这种机会，但是我却在反复建站中，丢失掉了那些本应属于我的机会——反反复复地建站，一步一步地摧毁自己的回忆，把自己的想法扼杀在摇篮里，换了一个又一个域名。

就像我往期博客里记录的那样，早年的第一个博客网站是跟着 S 佬搭建的，也就是上文提到的第一个站点。我现在还记得那篇文章——因为我根本就没有删，当时是使用 MkDocs 搭建博客网站。因为 MkDocs 本身只是个文档站引擎，网站上的一切东西都很简陋，当时建站的主要目的就是在网上装二次元（其实我更喜欢特摄），分享自己的 Linux 经历。

文章大致是这样子的，那天中考英语考试回来。我终于有时间折腾自己从拼多多买的二手 Matebook E 2022，这是一台由华为出品的二合一平板电脑。为它安装 Linux 可不是一件简单的事情，由于华为平板的驱动问题，网上出现了一大堆关于这类描写的文章。文章里有一张照片就是当时我安装好了 Arch，随手拍的 Linux 平铺桌面，虽然没解决驱动问题。

这就是为什么我那么执着于写博客，以至于一天多更都是常出现的情况，因为我以前就是写图文的。最开始是在 CSDN 上面写，至于博客园和其它类似的个人主页网站，我根本就没有用过。对我来说，CSDN 才是我萌新时期的回忆，无论是搜索 Python 问题还是搜索文档接口，百度上权重最高就是它。博客园和个站反而是后来用必应认识的，就是如此。

后来就是如此，我一边在华为平板上折腾 FydeOS，或者在自己的游戏本上安装 FydeOS。再把这些经历、踩坑，全部写成博客教程文章发到 CSDN 账号上。有时候我也会不知道写什么，因为对我来说，仅仅对我来说，这时候的博客仅仅只有教程和所谓的技术解析。离开了这些东西，我压根不知道原来博客可以写生活，或许这就是那天小芬来跟我吐槽不想写博客的原因。

去年的十月份，离开了技校。在网上开了一个技术小店，来的人确实挺多的。很多时候都是忙的要死，但是却收很便宜的钱。那时是我最开心的时候，因为我每天都能收到超过一百块的零花钱，我知道那些钱是自己挣来的，反而花的很舒服。但是后来我却关停了它，仅仅只是几天。后来小店人少了，冷清了。我的零花钱少了，以前至少还能每周转两百给我爸。

那段时期我一样有一个博客网站，我记得是凌晨的时候在抖音发了图文。当时是这样写的：“深夜是写作最好的良药！”，配图就是我在搭建博客网站（依旧 MkDocs）。甚至这个网站也不是打算建给自己的，而是建好放在闲鱼上卖的（虽然相当于没建就是了）。但是评论区有个人和我私信交换友链（我以前是不知道写博客有友链的），我也就换到了第一个友链——小小小韩。

后来接触了 Jekyll 的 Chirpy，我才知道原来建站真的可以那么方便。原来建站的 RSS 真的可以是主题自己给的，原来评论区功能真的可以是主题自己内置的。RSS、友链、关于……这些词在我的脑海徘徊了很久，我认识到了很多的新词汇。这时已经是 2025 年的 12 月了，我用着一个简单到不能再简单的博客网站，在 B 站录制自己的教程视频。

对于当时的我来说，所谓的博客和视频，本来就是分不开的。甚至我还有闲心在游戏本写完博客，换到二手 MacBook Air 上录制讲解。只因曾经在游戏本上录制视频，好多人说我的麦像是炸了一样，所以我选择了那个二手 MacBook Air，因为它屏幕好、收音好。这就是为什么我后来把这俩卖了，买了个新的 MacBook Air，因为我几乎只玩《我的世界》和《泰拉瑞亚》。

与其说是博客、录教程，不如说更像是在整活。我根本没打算认真地记录自己，只写了教程之类的文章。写完这些知识文章，我就不知道要写什么了，整活博主的通病了。那之后吧，我得了一场持续了两个月的感冒，吃了很多药好好休息才好。听说是当时开店折腾太累，以及初中的时候长期熬夜折腾，算在一起的“累病”。那段时间，我话都说不了，鼻涕很多。

我能干什么持续下去呢？我订阅了 LKML，或许我可以发挥一下我的特长，比如顶着生病的腔调给大家读邮件？额，那还是算了。LKML 的更新几乎成了我视频题材的稻草，我当时真的想的是如果我能一直干下去，或许我就可以熬到 18 年岁的时候，在我的网站上搞满广告，给自己带来收入（现在的我当然不会）。就像是我以前学习的时候，看的那些粗制滥造的中文翻译网站。

今年一月的时候，我把持续了两个月的博客网站翻新了一下。如何翻新的呢？我当时从 Jekyll 换到了 Hexo，主题从 Chirpy 换到了 Butterfly，再把文章保留下来，这就是翻新，仅此而已。我真的只是用 Butterfly 主题，建了一个丑丑的“教程站”。当时的我依旧模仿那些二次元博主，搞什么“梁栋烨的 Lifeboat”之类的称呼，现在想想就好笑。

后来我又搞了不知道多少“个站”，用过的 Hexo 主题都能从这头排到那头了。什么 Butterfly、Cactus、AnZhiYu、ZenMind、Landscape……我就记得峰哥有一句很好笑的话，他说我的博客网站像是开往，每天点同一个域名，总能来到不同地方。那时还没有认识峰哥，但是已经开始和一些人交换友链了，交换的方式很粗暴，就是从王科文友链页点进去一点一点交换。

我的网站几天换一次，因为当时没有域名。我的 GitHub 名字没几天换一次，网站地址自然也跟着换。王科文每次到周末，就会检查一下自己的网站友链，在自己的评论区问我网站怎么回事。我只能尴尬地回答他以上理由，因为当时还只是个萌新。现在终于知道换友链为啥至少运行一个月了，至少人家能运行一个月，不是真运行就是想烂在那了，至少也能访问。

网站名字也跟着换啊，从什么“梁栋烨的 Lifeboat”、“夏双菊的土地”、“梁栋烨的博客”……一直换到快过年，我才买了自己的第一个域名。域名买的 `.top` 域名，只需要 18 块。虽然不是什么高大上的域名，但好歹也算是有一个互联网地址了，别人从这个地址可以直接进来。快过年的时候，用 EMLOG 搞的站点，花了 81 块，网站名叫“临久”。

一直写一直写，写到过年的时候，我都一直在写生活。现在想起来，这两个时期真的是两个极端。一个是极端的整活教程，一个是极端的记录生活。当时的我是会没搞懂一件事，代码从来都是我人生的一部分，哪有什么粗鲁地分为“生活站”、“技术站”，网站当然都可以什么都写，只要自己乐意就行。当时的我依旧是开始写博客，缺乏关注的。

当时的我真的很希望博客网站每天都有人来，所以过年的的时候我去别人网站评论，希望能让别人想起来还有个站没来。过年的时候认识到了十年之约，这是一个博客活动，很多人都参与了。网站要求至少是一年起步，语文不太好的我试着把自己的网站也提交进去。果然，过了新年，我的网站没加入进去。不过也是知道那么个活动就是了，或许我可以直接来？

当时的我也是傻啊，其实最简单方式就是换个友链。交换友链以后，大家就是朋友了，会有时间来看看我的，可是我却不知道。但或许正是因为这几个月以来的不断学习，能让我意识到当初的自己有多么愚蠢，我可不想做那种一辈子坐在椅子上傻乐的人。至少还是要有点自我意识和学习能力的，或者想象力。别让生活处于太无聊或者太动荡，就在它们之间就行。

不出所料的，“临久”网站没过多久也没了。我又搞了个 WordPress 的网站，这次是真的可以用我最爱的主题了。WordPress、Ghost、EMLOG、Memos 这些动态站引擎，陪伴我一段时光。那时我第一次尝试在一个网站，同时写技术、生活、说说。也算是本博客更新模式最早的雏形了，因为没换过友链，所以网站依旧没有人。自己一个人偶尔写写东西，也挺好。

除了 WordPress 的站点，我还同时维护着一个静态的文档站点。这次我拿我的 WordPress 给文档站当图床用，解决了 GitHub 网站速度慢的问题（虽然现在知道了可以通过配置 Cache Rules 来解决）。这次我的文档站终于凑够了 90 篇文章，这对我来说简直相当于里程碑意义的，那可是 90 篇文章诶，还是一个月 90 篇。不妨想想，如果是一年呢？

后来嘛，发生了两件事，让我记忆犹新的，都不是好事。有一天，我闲来无事查看了树莓派的网络日志，发现一个月就用了近 200 多 GB 的流量——我的树莓派的 WordPress 不仅承担图床，同时还运行着网站的评论区，当时我不知道 Vercel 可以部署 Waline，也不知道 Waline 可以导出评论，导致好友来我站点的评论都没了。

这件事之后，使我更加下定决心使用 Serverless 和静态页面网站。虽然我有服务器，但那是对于内网的我而言的，如果再做一个之前做过的事情，指不定哪天又中招了。一个月更新了一百篇文章，对于爬虫来说简直就是大餐，无数的访客蜂拥而至，这种情况下，我却没有做好应对措施。能对付他们的，只有欠费的宽带，和网站没了的我。

买了新电脑后，网站后台没了，换了一个博客引擎。这次又用回 Chirpy 了，评论区又用回了 Giscus，也还行吧！过不了几天，又换回了 Hexo，这次用的是周良开发的 ZenMind 主题。再之后在卡泽的影响下，我又用回了这个老朋友——Butterfly。也开始有了一颗知道“从现在开始”的心，网站的名字定格在了简单的“他说”，用户名直接用本名。

这一路过来，有太多值得后悔的事情了。像是初中的长期休学，导致最后中考成绩差只能考技校。觉得技校里面不干净，于是离开了技校。这对于大多数人来说，这些事情都值得后悔半辈子了。但对我来说，这并不值得我后悔。因为我知道，哪怕我初中在校，最后成绩也会因为长期压力大而考差。如果是因为这一点而考差，我反而会更难过。

离开了技校，或许是正确的。这是建立在我初中休学期间已经学会了一项技能，并且认识到技校真实环境的情况下。我们这的“技工学校”，就是那种两百多分就能考上的垃圾学校，来这读书的人大部分都不是奔着读书来的。最让我感觉庆幸的那一次，是在退学的一个月后，前技校班主任打来电话，说班上一个同学得肺结核。我真庆幸离开了那里，没有得病。

如果是从去年开始，一直写博客到现在，没准我真的可以参加那些博客活动——十年之约、博友圈。再不济，如果是从今年开始写博客到现在，至少也过两个月了，可以参加开往了。但或许这也不是好事，原因有几点：如果从去年开始写到现在，当时的我还没有三观成熟，会写很多让今天的自己更后悔的暴论；如果是从今年开始写的，那些粗制滥造的 AI 水文并不值得。

CSDN 很差，在中文圈子里被口诛笔伐。原因是它过度地商业化了，网站像是没有钱一样。原本一个好好的技术站，在如今的环境下居然做成了广告堆。里面的大部分内容都是 AI 写的，有些内容一眼就不是作者自己原创，可是如果你急用的话，往下一翻还有让你更急的——0.1 块钱每天解锁文章。但这并不妨碍我写在 CSDN 的教程被一些开源社区收录，平台只是载体。

最近我又去看了卡瑟布尔的博客，他写了一篇新的碎碎念。这次的内容比较涉及隐私，需要密码才能看。密码在提示里说，是他的生日。作为他的朋友，自然是记得他的生日是几号的。看了一眼内容，大多是在后悔、怀念、感悟一些什么东西。观有所感，这勾起了我的回忆，让我想起来写这篇文章。人的一生只有一百年，算下来也就三万天。

就像是现在一年，也不过就是你一睁眼、一闭眼的样子。说是一百年，实际算下来很快的。我的宠物龟们，它们的寿命也不过才 20 来年，几乎就是我养它们大，再养它们老的程度了。或许对于人来说，乌龟就是这辈子养过的其中一只宠物；但对于龟来说，人就是龟这辈子的全世界。我们不敢改变过去什么，也不能做什么。唯一能做的就是向前看，努力做下一件事不让自己后悔。
